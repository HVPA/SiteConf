Prerequisite:
- Python 2.6+
- Django 1.4+
- TastyPie 0.9+
- MySQL 
- Apache 2
- WSGI

SiteConf Django app:
SiteConf is a REST webservice used to support VariantExporter client application for labs to export clinical data.
The VariantExporter applications configuration settings, gene/disease information and gene tagging information are stored 
on the SiteConf database.

Setup:
Please see Django documentation for deploying the django app at https://docs.djangoproject.com/en/dev/howto/deployment/
